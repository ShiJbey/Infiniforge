Infiniforge REST API

Routes

GET: /help

Opens this page with information about available routes


GET: /tools/sandbox

Opens the live web GUI that interacts with the active server

GET: /tools/crosssection/ < name >

Returns the json json representation of the cross section with the given name

GET: /tools/crosssection

Opens the live web GUI for editing and saving blade cross secetions


POST: /api/crosssection/ < name > / < jsonData >

Saves the json representation of the cross section to the json file with all the cross sections.

GET: /api/forge/sword/style/ < style >

Forge a sword with the given style

GET: /api/forge/sword/style/ < style >/options/ < options >

Forge a sword with the given style and with the additional options passed. (See GenerationParameters in docs.)

GET: /api/forge/sword/style/ < style >/seed/ < seed >

Forge a sword with the given style and seed

GET: /api/forge/sword/style/ < style >/seed/ < seed >/options/ < options >

Forge a sword with the given style and seed and options. (See GenerationParameters in docs.)
